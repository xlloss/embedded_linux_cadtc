Memory Zones

  • Normal memory
    • GFP_KERNEL
  • DMA-capable memory
    • __GFP_DMA
    • Platform dependent
    • First 16MB of RAM on the x86 for ISA devices
    • PCI devices have no such limit
  • High memory
    • __GFP_HIGHMEM
    • Platform dependent
    • > 32-bit addressable range
