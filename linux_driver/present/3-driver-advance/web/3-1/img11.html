1. 前面主要是使用 kmalloc() 向系統請求記憶體. 這種方式比較偏向記憶體體積較小的應用.

2. 如果請求的記憶體體積較大, 需要使用 get_free_page() 相關的 kernel API.

3. 已 get_free_page() 類似的方式請求記憶體, 如 kernel API 名字所述 - 以 Page 為單位. 1 page = 4K Byte.

4. 另外 __get_free_page() 只能拿一個 page.

5. __get_free_pages() 可以拿超過 1 個 page. 單位以 2^order 為單位.

    例如 :
    order = 0 拿 1 page = 1* 4K-byte.
    order = 1 拿 2 page = 2 * 4K-byte.
    order = 2 拿 4 page = 4 * 4K-byte.
