The Flags Argument

  • GFP_USER
    • is used to allocate user pages; it may sleep
  • GFP_HIGHUSER
    • allocates high memory user pages
  • __GFP_DMA
    • requests allocation to happen in the DMA-capable memory zone
  • __GFP_HIGHMEM
    • indicates that the allocation may be allocated in high memory
