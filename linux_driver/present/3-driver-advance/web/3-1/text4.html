The Flags Argument

  • kmalloc prototype
  • GFP_KERNEL
    • is the most commonly used flag
    • Eventually calls __get_free_pages (GFP prefix)
    • Can put the current process to sleep while waiting for a page in low-memory situations
    • Cannot be used in atomic context
