Messages Control

  • Need to recompile to turn messages on and off
    • Faster than using C conditionals
  • In a header file (e.g., scull.h)
  • #undef PDEBUG /* undef it, just in case */
  • #ifdef SCULL_DEBUG
  • # ifdef __KERNEL__
  • # define PDEBUG(fmt, args...) \
  • printk(KERN_DEBUG “TEST: “ fmt, ## args)
  • # endif
  • #endif
