1. Minor number
一支單獨 Linux device driver 如何管理不同的硬體資源 ?
例如 : SOC 上基本是會有一組以上的 SPI bus.
但是 SPI bus driver 基本上只會有一支.
一個 SPI bus driver 如何去管理不同的 SPI Bus.
靠的是 "Minor number"


2. user-land program 開啟不同 input device driver. 我們可以想像它為不同的 Keyboard driver.
每個 keyboard 都是相同的 input device driver 管理.
因此 input device driver 藉著 Minor number 去辨識管理那一個硬體資源 [ keyboard ]


3. device driver :
     1. Major Number : 用來辨識哪個 device driver
     2. Minor Number : 用來辨識哪個 hardware device
