一個簡單的 Linux 系統環境圖. 主要包含 user-land 的 Application,
透過 Linux kernel system call API 與 kernel-land 的 driver 做互動.


其中 Char Device Driver 就是我們的講解主題. Character Device Driver (字元驅動程式) 字元驅動程式透過檔案串流的方式與 user-land 的 Application 進行溝通.
例如:
    File_Open(), File_Read(), File_Write(). IOCTL().


之所以會有 File_XXX() ... 即是 Linux system 的特性之一.
Linux 會將許多 kernel-land 的控制透過虛擬檔案系統的管理與 user-land 的 Application 做溝通.
大大簡化處理的方式.
