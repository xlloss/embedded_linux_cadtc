1. 在 exynos4412-tiny4412.dts 中, 我們需要使用 PWM 功能.

2. exynos4412 SOC 中有兩個 Pin 可以使用 PWM 功能. 我們需要去決定要用哪一個.

3. 從 exynos4412-pinctl.dts 中可以找到 pwm0, pwm1 這些 node.

4. 從 exynos4412-pinctl.dts 的 node 中可以看到其對應的 Pin 腳位

5. PWM0:
    HW Pin Name: gpd0-0
    Pin function Select : EXYNOS_PIN_FUNC_2

6. PWM1:
    HW Pin Name: gpd0-1
    Pin function Select : EXYNOS_PIN_FUNC_2
