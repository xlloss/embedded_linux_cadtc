Devicetree and Tiny4412

  • Enable Device Tree Support
    • menuconfig → boot options → flattened device tree
    • USE_OF select by Y (default Y)
  • Build the Devicetree
    • $KERNEL/script/dtc
    • $make dtbs
    • $KERNEL/arch/arm/boot/dst/
      • Exynos4412-tiny4412.dtb
  • Boot Linux kernel
    • #bootm KERNEL_ADDR – DEVICETREE_ADDR
    • #boot KERNEL_ADDR (use DTB append kernel)
